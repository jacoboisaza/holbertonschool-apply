Some basic Cheatsheets

Linux

  • man command ::Manual
  • pwd ::PrintWorkingDir
  • ls -lahZ ::List LongAllHumanContext
  • cd ::ChangeDir
  • less file1 ::Print file content
  • touch tile2 ::ModifyFileTime or CreateFile
  • cp from to ::Copy
  • mv from to ::Move
  • mkdir dir1 ::MakeDir
  • rmdir dir2 ::RemoveEmptyDir
  • rm -r ::RemoveAllDirectoryContentsRecursively

HTML5

  • <h1> - <h6> ::Heading
  • <p> ::Paragraph
  • <i> ::Italic
  • <b> ::Bold
  • <a> ::Anchor
  • <ul> & <li> ::Unordered List & List Item
  • <blockquote> ::Blockquote
  • <hr> ::Horizontal Rule
  • <img> ::Image
  • <div> ::Division

Emacs

  • C-x C-s ::save a file back to disk
  • C-x C-c ::exit Emacs permanently
  • C-s ::search forward
  • C-r ::search backward
  • M-p ::select previous search string
  • M-n ::select next later search string
  • M-g g ::goto line
  • C-k ::Line killing
